Project: AI-Powered Procurement & Spend Intelligence 🚀
Objective: To provide procurement leaders with a diagnostic tool that identifies leakage, evaluates buyer efficiency, and uses machine learning to predict spend influencers.

Tech Stack:

Visualization: Power BI (Advanced DAX for Waterfall & Sankey charts) 📊

AI Integration: Power BI "Key Influencer" and "Decomposition Tree" visuals. 🤖

Data Modeling: Star schema connecting Purchase Orders, Accounts, and Departments. 🗄️

Key Dashboard Features:

Spend Waterfall: A Q1-Q4 analysis showing the growth trajectory to an $87M total annual spend. 📈

Sankey Flow Diagram: Visualizing the complex relationship between Domestic/International sourcing and Receiving Departments. 🔄

Key Influencer AI: Automatically identifies which categories (like T&E) cause the most significant drops in PO values. 🔍

Purchaser Efficiency: Comparing PO Amount vs. Saving Amount for individual buyers to identify top negotiators. 🤝
